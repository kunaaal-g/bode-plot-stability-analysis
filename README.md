# Stability Analysis using Bode Plot

## Overview

This repository contains the stability analysis of a control system using the frequency response approach (Bode Plot method). The analysis includes theoretical derivation, numerical calculations, and MATLAB implementation.

## System Transfer Function

G(s) = 40 / [s(s+2)(s+20)]

## Objectives

* To understand frequency response of systems
* To analyze stability using Bode plots
* To determine Gain Margin and Phase Margin
* To validate results using MATLAB

## Contents

* `report/` → Detailed technical report 
* `matlab/` → MATLAB code for Bode plot and margin calculation
* `images/` → Generated Bode plot

## MATLAB Implementation

The MATLAB script generates:

* Bode plot
* Gain Margin
* Phase Margin
* Crossover frequencies

## Key Results

* Gain Margin ≈ 27 dB
* Phase Margin ≈ 63°
* System is Stable
